Shared Memory

Nothing to set up
Assetto Corsa and Competizione publish telemetry to shared memory whenever they're running. There's no port to choose and nothing to enable in-game — start the bridge and GridGlow picks the game up when you launch it.
Same PC only
Shared memory is local to one machine, so unlike the other titles GridGlow has to run on the PC you're racing on. UDP forwarding doesn't apply for the same reason.
